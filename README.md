# AviUtlAutoInstaller
AviUtlの動画作成環境を自動で行うツールです。

## 注意
このツールは無保証で提供されます。  
このツールを使用したことで発生した全ての損害に関して保証しません。

## 使い方
1. AviUtlAutoInstaller.exeを実行
1. インストール先を選択
1. インストールボタンを押して終了まで待つ

## 開発環境
* WSL(Pengwin)
* Windows版 Python(v3.7.7)
* Windows版 PyInstaller(v3.6)

### ビルド方法
* リリース  
`$ make`
* デバッグ  
`$ make debug`
* クリーン  
`$ make clean`
* 実行  
`$ make run`
